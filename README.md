# Seidor System

Aplicação fullstack de gestão de funcionários desenvolvida em Node.js, dividida em **server (Expressjs)** e **client (React)**.

## Instalação das dependências

O projeto possui dependências separadas para **server** e **client**.

Para instalar tudo **de uma vez**, no diretório raiz:

```bash
npm run install:all
```

Para rodar o servidor, execute no diretório raiz:
```bash
npm run server
```

Para rodar o client, execute no diretório raiz:
```bash
npm run client
```

As aplicações serão iniciadas em http://localhost:5000 e http://localhost:5173, respectivamente.
